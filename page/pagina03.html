Cesta Date

Esta cesta é uma opção perfeita para aqueles que apreciam um bom vinho e chocolates deliciosos. A cesta inclui uma garrafa de vinho de alta qualidade, escolhida especificamente para complementar os chocolates selecionados.Essa cesta de vinho é uma ótima opção para presentear em diversas ocasiões, como aniversários, Dia dos Namorados, Dia das Mães, Natal e outras datas especiais. É uma maneira elegante e deliciosa de mostrar a alguém que você se importa.

 

Cesta Casual

Essa cesta é perfeita para quem quer desfrutar de um momento delicioso e casual com tudo que é necessário. Ela inclui uma garrafa de vinho escolhida especialmente para combinar com a seleção de frutas frescas e doces, como pera, maçã e chocolate. Além disso, a cesta inclui queijo selecionado, criando uma experiência gourmet completa.

 

Cesta Picknick

Esta cesta é ideal para quem quer fazer um lanche agradável como um picknick em um parque.contendo uma seleção fresca de frutas e queijos. A cesta inclui uma garrafa de vinho selecionada para complementar as frutas frescas, como uvas, pera, maça verde e outras frutas da estação. Além disso, a cesta inclui uma variedade de queijos selecionados, para uma experiência completa e deliciosa.

 

Cesta Brunch

Esta cesta é perfeita para quem ama um brunch completo e relaxante. A cesta inclui uma garrafa de espumante, ideal para criar uma atmosfera de celebração. Além disso, a cesta inclui itens para um café da manhã completo, como pão, frios, chocolates, biscoitos e outras delícias matinais. É uma opção perfeita para presentear alguém especial ou para desfrutar de um momento delicioso com amigos e familiares.
